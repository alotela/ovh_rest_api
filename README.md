# OvhRestApi

This is a gem to interact with OVH RESFTULL API 1.0
https://api.ovh.com

## Installation

Add this line to your application's Gemfile:

    gem 'ovh_rest_api'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ovh_rest_api
